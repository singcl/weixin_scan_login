# 微信公号扫码登陆&小程序扫码登陆-NEST

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit test
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
